# Secure Contacts App (SCA) Connectors

This repository contains the officially supported connectors for the **Secure Contacts App (SCA)**.

SCA Connectors enable the integration of external business systems with Secure Contacts App, allowing organizations to securely synchronize and manage business contact data from a variety of enterprise sources.

---

## Available Connectors

| Connector | Description |
|------------|-------------|
| **HubSpot Connector** | Synchronizes contacts from HubSpot CRM into SCA. |
| **Salesforce Connector** | Synchronizes contacts from Salesforce into SCA. |
| **SAP Connector** | Synchronizes business partner and contact data from SAP S/4HANA into SCA. |
| **AAD Cache Connector** | Provides Microsoft Entra ID (Azure AD) user data through the SCA AAD Cache service. |

---

## Downloads

Each connector folder contains the latest connector package (`.zip` file), including:

- Connector binaries
- Installation guide
- Configuration documentation

To get started:

1. Open the folder of the desired connector.
2. Download the connector package (`.zip` file).
3. Extract the package.
4. Follow the included installation and configuration guide.

## About Secure Contacts App

Secure Contacts App (SCA) is an enterprise contact management solution that enables organizations to securely manage, synchronize, and distribute business contacts while maintaining full control over corporate contact data.

Learn more at [Secure Contacts App](https://secure-contacts.com).
