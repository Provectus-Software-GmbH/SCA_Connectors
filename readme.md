# Secure Contacts App - Connectors

This repository contains the available **SCA Connectors** and their corresponding implementation folders.

## Available Connectors

The repository currently includes the following connectors:

- **SCA HubSpot Connector**  
  Contains the integration logic and documentation for connecting SCA with HubSpot.

- **SCA Salesforce Connector**  
  Contains the integration logic and documentation for connecting SCA with Salesforce.

Each connector is maintained in its own folder and can be used independently.
